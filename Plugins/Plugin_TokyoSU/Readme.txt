tr1_enemies and EnemySlots are enemy packs made by LaraHCroft91 containing TR1 and TR2 enemies that can be used with this plugin, respectively.

list_of_objects lists all enemies and the levels in which they can be found.

This plugin requires that Plugin_ObjectHelper is installed.

Install instructions:
1) Launch NG Center
2) Go to [Plugins]
3) Click on [Install New Plugin] and select this folder
